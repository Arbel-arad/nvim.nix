# Arbel's neovim flake

## Usage

```bash
nix run git+https://forgejo.spacetime.technology/arbel/nvim.nix
```

GUI version:

```bash
nix run git+https://forgejo.spacetime.technology/arbel/nvim.nix#gui
```
